Class Detail

Pushpop.PickerTableViewDataSource(dataSet, defaultReuseIdentifier)
Creates a new data source for a PickerTableView. NOTE: This is not to be used as a data source for the list of items to select from. This is to be used by a PickerTableView to store the list of items that have been selected.
Parameters:
{Array} dataSet Optional
An optional array of data to initialize a default data source.
{Array} dataSet.id Optional
The unique identifier for a specific record.
{Array} dataSet.value Optional
The (sometimes) hidden value for a specific record.
{Array} dataSet.title Optional
The title to be displayed in a TableViewCell for a specific record.
{String} defaultReuseIdentifier Optional
The optional reuse identifier to be used for rows that do not specify a specific reuse identifier.

{Pushpop.TableViewCell} getCellForRowAtIndex(tableView, index)
Returns a TableViewCell for the specified index. NOTE: This is the default implementation and should be overridden for data sources that are not driven directly from an in-memory data set. Also, in order to account for the picker cell, this method should always return the picker cell for the last available index.
Parameters:
{Pushpop.TableView} tableView
The TableView the TableViewCell should be returned for.
{Number} index
The index of the data to be used when populating the TableViewCell.
{Number} getNumberOfRows()
Returns the number of rows provided by this data source. NOTE: This is the default implementation and should be overridden for data sources that are not driven directly from an in-memory data set. Also, in order to account for the picker cell, this method should always return the number of rows in the data source plus one.
{Number} getPickerCellIndex()
Returns the index for the "picker" cell. This is used to differentiate the "picker" cell from the other cells representing the list of selected items. NOTE: The "picker" cell is the cell that will push a new table view with a list of items to select from when it is selected.
{String} getPickerCellReuseIdentifier()
Returns the default reuse identifier that this data source will use when a reuse identifier is not specified in the data for the "picker" cell. NOTE: The "picker" cell is the cell that will push a new table view with a list of items to select from when it is selected.
{String} getPickerCellTitle()
Returns the title to be displayed for the "picker" cell. NOTE: The "picker" cell is the cell that will push a new table view with a list of items to select from when it is selected.
