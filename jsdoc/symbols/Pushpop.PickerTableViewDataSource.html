An optional array of data to initialize a default data source.

The unique identifier for a specific record.

The title to be displayed in a TableViewCell for a specific record.

The optional reuse identifier to be used for rows that do not specify a specific reuse identifier.

{Pushpop.TableViewCell} getCellForRowAtIndex(tableView, index)
REQUIRED: Returns a TableViewCell for the specified index. NOTE: This is the default implementation and should be overridden for data sources that are not driven directly from an in-memory data set. Also, in order to account for the picker cell, this method should always return the picker cell for the last available index.
Parameters:
{Pushpop.TableView} tableView
The TableView the TableViewCell should be returned for.
{Number} index
The index of the data to be used when populating the TableViewCell.
{Number} getNumberOfRows()
REQUIRED: Returns the number of rows provided by this data source. NOTE: This is the default implementation and should be overridden for data sources that are not driven directly from an in-memory data set. Also, in order to account for the picker cell, this method should always return the number of rows in the data source plus one.
